## Merge two BSTs into a doubly-linked list in sorted order

**Given two binary search trees, merge them into a doubly-linked list in sorted order.**

    For example,

    Input: Below BSTs

          20
         /  \
       10    30
            /  \
           25  100

          50
         /  \
        5    70


    Output: Below DDL

    5 —> 10 —> 20 —> 25 —> 30 —> 50 —> 70 —> 100 —> null
